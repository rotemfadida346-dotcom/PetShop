## Testing Image Load

### Check if images work:
1. Open product page
2. Images should load instantly
3. Should see colored squares with product names
4. Thumbnails should switch main image

### If images don't load:
1. Check Next.js config (remotePatterns)
2. Check network tab in browser
3. Verify URLs are correct
4. Try rebuilding: `npm run build`

## Optimization Tips

### For Real Images:

1. **Compress before upload:**
   ```bash
   # Using imagemagick
   convert original.jpg -resize 800x800 -quality 85 optimized.jpg
   
   # Online tools
   - tinypng.com
   - squoosh.app
   - imageoptim.com (Mac)
   ```

2. **Use WebP format:**
   ```bash
   convert original.jpg -resize 800x800 optimized.webp
   ```

3. **Lazy loading:**
   - Already implemented with Next.js Image
   - Only loads when scrolling into view
